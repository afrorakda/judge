# #74 Judge

A minimalist binary decision-making tool designed to eliminate "Decision Fatigue." No animations, no fluff—just an instant, cold verdict to help you move forward.

## Core Philosophy
- **1 Tool / 1 Action**: Tap the button, get your answer. Zero delay.
- **Decision Integrity**: Prevents re-rolling by locking the interface for 3 seconds after each verdict. 
- **The Verdict is Final**: Encourages users to commit to the result rather than playing with "odds."

## Features
- **Instant Result**: Displays "YES" or "NO" in monolithic typography the moment you tap.
- **Visual Feedback**: A subtle "flash" effect on tap ensures you know a new decision has been made.
- **Auto-Reset**: Returns to the neutral "?" state after the 3-second lock, maintaining a clean slate for the next crisis.
- **High Contrast UI**: Uses the signature Fridge Combo #333 and #fff palette for maximum authority.

## Design System
- **Background**: #fff
- **Main Color**: #333
- **Border Radius**: 15px
- **Dynamic Scale**: The verdict text expands to 8.5rem to command your attention.

## Privacy & Performance
- **No Login**: No barriers to your decision.
- **No Tracking**: Your dilemmas stay in your head; no data is ever sent to a server.
- **No Ads**: A distraction-free environment for serious choices.
- **Browser Only**: Ultra-fast, responsive, and optimized for mobile/Safari.
